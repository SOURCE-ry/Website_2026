# Roadmap

## Phase 1 - Project setup

- Add README, ROADMAP and CHANGELOG
- Add licence (MIT)
- Define folder structure:
    - Root
        - public
            - fonts
            - images
            - favicon.ico
            - manifest.json
            - index.html
        - src
            - components
            - pages
            - styles
            - utils
- Add .gitignore

## Phase 2 - HTML structure

- Use semantic HTML structure
    - Header
    - Navigation
    - Main content
    - Sections
    - Footer
- Ensure correct heading hierarchy
- Add meta tags
- Add page titles and descriptions
- Implement consistent layout across pages

## Phase 3 - CSS & Responsive design

- Set up global styles
- Define design system
- Use mobile responsive design
- Implement layout system (flexbox, grid)
- Ensure consistency across pages
- Test Layout on multiple screen sizes
- Optimize for readability and clarity

## Phase 4 - Accessibility

- Use semantic elements
- Add alternative text for all images
- Ensure sufficient color contrast
- Make all functionality accessible via keyboard
- Provide visual focus indicators
- Test with screen reader basics
- Validate against WCAG guidelines
